# 航迹起始 | TrackInit - 多目标



# 计划
最终封装为一个完整的类，支持事务处理，能够接收一批点迹数据，输出一批航迹起始信息。支持多种算法选择和参数配置。

## 架构设计
### 计划写的算法
1. **基础算法**  (先拿来比较下看看效果)
   - M/N逻辑确认
   - 最小二乘状态估计
   - 物理约束过滤

2. **霍夫变换家族** (开发中)
   - 标准霍夫变换
   - 概率霍夫变换
   - 约束霍夫变换

3. **RANSAC系列** (计划中)
   - 基本RANSAC
   - 多模型RANSAC
   - 自适应RANSAC

4. **机器学习增强** (实验性)
   - 基于ONNX的轨迹验证
   - 特征工程工具

## 软件框架
### 1. 结构体及不可变参数定义
  - def_init.h 用于定义错对外结构体，系统参数等
### 2. 抽象层
  - TrackInitBse 用于定义所有航迹起始组件的接口
### 3. 组件层
   - HoughSlice 基于DBSCAN的霍夫变换算法，将目标区域基于聚类切片，对每个聚集区域，基于霍夫变换执行航迹起始
### 4. 工具层
   - ObjectPool 通用对象内存池
   - Func_dbscan 聚类算法实现 
### 5. 服务层
   - InitService 航迹起始服务层，占据一个单独的线程，调用接口对外提供服务
